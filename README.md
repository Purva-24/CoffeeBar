# ☕ CoffeeBar

> A clean and responsive cafe website built with pure HTML & CSS — explore the menu, book a table, and pre-order your favorites!

---

## 📌 Project Overview

**CoffeeBar** is a static cafe website built using only HTML and CSS. It showcases all the items available at the cafe, provides detailed information about each one, and includes pages for table booking and pre-ordering. The project focuses on clean design and a smooth user experience without any backend or JavaScript frameworks.

---

## ✨ Features

- 📋 **Menu Page** — Browse all available cafe items with their details and prices
- 🪑 **Table Booking** — A form to reserve a table for your preferred date and time
- 📦 **Pre-Order** — Select items and place an order before your visit
- 📱 **Responsive Design** — Works well across different screen sizes

---

## 🛠️ Built With

- **HTML5** — Page structure and content
- **CSS3** — Styling, layout, and responsiveness

> No JavaScript. No frameworks. No backend. Pure HTML & CSS only.

## 🚀 How to Run

No installation or setup needed!

1. Download or clone the repository
2. Open `index.html` in any web browser
3. That's it — the site is live locally!

```bash
git clone https://github.com/your-username/coffeebar.git
cd coffeebar
# Just open index.html in your browser
```
